# Youtube Downloader 😉

### Youtube Downloader With Inline YouTube Search Support ♦️

## Prerequisite
    ffmpeg
  
    
## install dependencies
    pip3 install -r requirements.txt


## Setup Bot
    - Change configuration config.py  File
    - install dependencies
    - python3 -m bot
